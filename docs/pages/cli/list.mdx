# List command
The list command is used to display the list of collections, requests, and environments.

### List Collections
- `tc list` to display the list of all collections
<br/>
<img width="650" alt="List Collections" src="/images/cli/list1.webp" />


### List Requests
- `tc list 'colName'` to display the list of requests in collection 'colName'
<br/>
<img width="650" alt="List Requests" src="/images/cli/list2.webp" />


### List Environments
- `tc list env` will display list of all environments
