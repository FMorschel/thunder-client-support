# Get Started

Please find the documentation links for Thunder Client Features

#### Git-Sync
- The [Git Sync](team#git-sync) feature is useful for saving data to a Git repository for team collaboration.
- Please note that Git Sync is a `paid feature`. Refer to the [pricing](https://www.thunderclient.com/pricing) for more details

- You can [import collections](features/import) from `Postman`, `Insomnia`, and `Open API 3.0`
